# Cz_Control_v2
為TeamCz的管理系統                            
分為兩大程式                   
CzControl(管理系統)               
CzServer(伺服端及監控系統)                

# CzControl(管理系統)
進去後三個java檔為登入介面、註冊見面跟主介面	

Exception放置程式出現問題時跳出的例外	                      
Features放置功能選單	                      
Utils則放置每個java檔都可能用到的功能	                    
Resource則是放置圖片等等的資源	                  

# CzServer(伺服端及監控系統)	
Servers為主要檔案	              
由ServerThead來負責處理使用者Socket等等的資訊
